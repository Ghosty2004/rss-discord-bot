# Romania Super Stunt - Discord Bot

<h3>Set up:</h3>

First of all you need to have **NodeJS** installed, then follow the console commands bellow

1. `npm install`
2. `npm install typescript ts-node`
3. `ts-node .`

<h3>Available commands:</h3>

- `/gangs`
- `/serverstats`
- `/stats`
- `/togglelog`
- `/top`
